Alimentar mensalmente

Sem mexer em código.

Fluxo recomendado

  1. Abra este dashboard publicado no GitHub Pages.
  2. Clique em Adicionar PDFs mensais e selecione o relatório do novo mês.
  3. Confira se os indicadores atualizaram.
  4. Clique em Gerar HTML com dados.
  5. Substitua o arquivo index.html do GitHub por esse novo arquivo.

Assim você só alimenta com PDFs e não precisa editar gráficos, fórmulas ou layout.

Backup da base

Use os botões abaixo para baixar ou restaurar a base em JSON. Isso ajuda se trocar de computador ou navegador.

Pronto para GitHub Pages

Estrutura mínima para publicar.

Para colocar no GitHub, renomeie este arquivo para index.html, envie para um repositório e ative Pages em Settings → Pages → Deploy from branch → main → /root.

Depois, sempre que importar novos PDFs e gerar a versão com dados, substitua apenas o index.html.
